# Sales & Demand Forecasting for Businesses

## Objective
To predict future weekly sales using Machine Learning.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Dataset
Walmart Sales Dataset

## Machine Learning Model
Linear Regression

## Results
The model predicts weekly sales and is evaluated using:
- MAE
- MSE
- RMSE
